Udef


Inherits from: Object

 

Store synthdefs in a global pool and make them available to synths on any server by sending them to the server at boot time, before starting any synth.


SynthDefs created while the server is running are sent immediately. ServerReady ensures that they will be sent before starting any further instance of SynthResource. 


Class Methods


*new( name, ugenGraphFunc, rates, prependArgs, variants, metadata, server) : 

Create a SynthDef in the same manner as in SynthDef new and store it for loading to server.

The arguments are the same as those of the new method of SynthDef.

server defaults to Server.default. 

If the server is booted, the SynthDef is sent immediately to it, and any synths sent at the same time

wait until it is loaded before they start. 

If the server is not booted, the Udef will be sent as soon as it boots.

Udefs created for one server during one work session remain stored in class variable all of Udef.

They are always loaded immediately after the server boots on each subsequent session. 


Class Variables


all : MultiLevelIdentityDictionary.

Level 1 keys: the servers that have Udefs defined on them.

Level 2 keys: The names of the Udefs defined for this server.

values: The Udefs defined for this server. 

All Udefs for a server are sent to the server as soon as it boots.
