ChainLink


Inherits from: Object

Purpose


ChainLink is created by Function:schedEnvir. It encapsulates the Function sched algorithm and stores the onEnd function in order to enable chaining.


Usage


ChainLink instances are not meant to be created directly. They are created by Function:schedEnvir in order to enable Chain to chain them in succession using onEnd.
