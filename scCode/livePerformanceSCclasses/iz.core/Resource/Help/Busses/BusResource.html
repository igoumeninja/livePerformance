BusResource


Inherits from: AbstractServerResource : Resource : Object

Purpose


Encapsulate the creation and access of busses. Main purpose is to enable access of busses via symbols with methods: 


<aSymbol>.control;

<aSymbol>.audio;

<aSymbol>.index;


Where sending the above messages to the symbol aSymbol will create the bus if it does not exist, or access the bus if it already exists and is bound to aSymbol. 


Usage


Example: 


\default.play.map(\freq, \freq);


\ctl1.playFunc({ | freq = 400 | Out.kr(\freq.index, freq) });


\ctl2.playFunc({ Out.kr(\freq.index, LFSaw.kr(0.2).range(-200, 200)) });


\ctl3.playFunc({ Out.kr(\freq.index, SinOsc.kr(7).range(-20, 20)) });


\ctl1.set(\freq, 800);


\ctl3.free;


\default.releaseSynth(1);
