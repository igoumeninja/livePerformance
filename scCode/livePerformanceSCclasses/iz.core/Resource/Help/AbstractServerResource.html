AbstractUniqueServerObject


Inherits from: UniqueObject : Object

 

Pupose


An abstract superclass for all classes whose instances represents objects on a server. All classes that inherit from this class store their instances under the server to which these instances belong, following this scheme: 


If <Class> is the class of an instance, <key> is the main key (name) of the instance and <server> is the server where the server object represented by the instance resides, then the instance is stored under the path: [<Class>, <server>, <key>]


Usage


All instances of an AbstractUniqueServerObject subclass that reside on a server can be retrieved by the class method onServer. 

For example: 


BufferResource.onServer; // all BufferResources on the default server.  


Udef.onServer(Server.internal); // All Udefs ("SynthResourceDefs") on the internal server.


SynthResource.onServer(Server.local); // all SynthResources in the local server. 


Instances of subclasses of AbstractUniqueServerObject can access their server by the method server. 


\default.play.server; // Access the server of the Synth started by \default.play


Related Classes


Subclasses: SynthResource, PrivateBusSynth, BufferResource
