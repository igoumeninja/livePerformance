ArchiveLoader


[Inherits from: ListWindow : UniqueWindow : UniqueObject : Object]

 

(Will be remodeled to be a UniqueObject stored with name created from its path, and create load / save list windows and dialogs)


This class is under development. Its purpose is to create a list of sessions of objects stored on an archive and allow the user to load a session selected from that list. 

 

Planned interface: 


*new (path)

create an ArchiveLoader that reads/stores sessions to/from a file on path <path>.



ArchiveLoader(<path>); // create an ArchiveLoader that reads/stores sessions to/from a file on path <path> 


*open (path)


*save (path, objects)


load (sessionName)

save (sessionName, objects)
