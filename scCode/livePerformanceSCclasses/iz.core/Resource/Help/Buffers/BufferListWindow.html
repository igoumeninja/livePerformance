BufferListWindow


Inherits from: ListWindow : WindowResource : Resource : Object

Purpose


Create a window listing all BufferResource instances with their names, buffer numbers, file paths and number of samples, with a text field for searching in the list by name of buffer, and the ability to search 


Usage


To create buffer list window instance for  a server, create an instance of BufferListWindow with that server as argument. For example:


BufferListWindow(Server.local); // Create buffer list window for the local server


The default for the server argument is Server.default. 


Related Classes


WindowResource, ListWindow


Instance Creation


*new(server) : Create new instance of BufferListWindow for server. 

server: An instance of Server. Defaults to Server.default.
