UserPath


Inherits from: Object

Purpose


Provide an easy way to add the root path of the users SuperCollider directory  (~/Library/Application Support/SuperCollider/) to any path string.


UserPath("path string...");


Is equivalent to:


Platform.userAppSupportDir ++ "path string..."


But easier and shorter to type. 


Usage


UserPath("startup.rtf");


UserPath("startup.*").pathMatch;



Related Classes


Platform, PathName.


Instance Creation


*new(pathstring) Return pathstring prepended by Platform.userAppSupportDir.
