//:d doc


DocListWindow provides a way to execute parts of the code in a SuperCollider document through keyboard commands. 


These commands are in the form of sigle key strokes corresponding to characters, such as "a", "b" "1", "0" etc. 


The commands are active when the code snippet list pane at the lower half of the "docs" window pane has the focus. This focus is made visible by a red outline around the list pane. To activate the code-list pane on a document, type Cmd-1 when that document is selected (window of the document is topmost). 


To mark part of a document for execution by LiltDoc, precede it by a comment in the form of 


//:<k>


where <k> is the character of the key that you want to press to evaluate the code following the comment. 


Example: 


//:1 Start Sound 1


a = { PinkNoise.ar(0.1) }.play;


//:2 Stop Sound 1


a.free;


The code above defines two "code snippets" and binds them to (keyboard) keys 1 and 2. 


If the key for a code snippet is the character "!" then that snipped will be executed whenever the document is brought to the top. For example: 


//:! Boot default server if not running: 


if (Server.default.serverRunning.not) { Server.default.boot };


//: rest


Activation


DocPoller.start;


This installs keyboard command Cmd-1 to re-parse a document for snippets and bring up the list of snippets with their activation keys in the bottom list of the Document List Pane.
