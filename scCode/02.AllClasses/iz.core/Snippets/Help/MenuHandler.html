MenuHandler


Inherits from: Object

Purpose


Enable the easy addition or removal of menu items for any object. 


(Limited to MacOS X for the moment)


Usage


To enable any object (Class or instance) to add menu items, define a class or instance method menuItems for it. 

The method should return an array of CocoaMenuItems. These items are stored in MenuHandler using the object (or class) as key, so that they can be removed when sending the object the message removeMenu. 


Example: 


(From Dock class, abbreviated:)


The Dock class defines the following menu items: 


*menuItems { ^[

CocoaMenuItem.addToMenu("Utils", "show doc list window", ["\"", false, false], {

this.showDocListWindow;

}),

CocoaMenuItem.addToMenu("Utils", "browse / open user classes", ["b", true, true], {

this.browseUserClasses;

}),

CocoaMenuItem.addToMenu("Utils", "open / create class help", ["D", true, false], {

this.openCreateHelpFile;

}),

CocoaMenuItem.addToMenu("Utils", "insert class help template", nil, {

this.insertClassHelpTemplate;

})

]

}

 

These can be added by evaluating: 


Dock.addMenu


and removed by calling: 


Dock.removeMenu


Related Classes


See also following class extensions: Object:addMenu, Object:removeMenu, Meta_CocoaMenuItem:addToMenu
