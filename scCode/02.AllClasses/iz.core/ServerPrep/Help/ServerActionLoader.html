SynthLoader


Inherits from: ServerActionLoader : Object

Purpose


Provide an algorithm for loading a set of objects to the server which ensures that all objects will be loaded before the next set of object is loaded, and which works even if new objects are added to the set while the loading is taking place. 


Usage


Used by ServerPrep. Not to be used directly. 


Related Classes


ServerPrep, DefLoader, BufLoader, RoutineLoader, SynthLoader
