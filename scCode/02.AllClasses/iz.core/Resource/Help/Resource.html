Resource


Inherits from: Object

Purpose


Provide a way to create an object when it is needed, and to access through a key it if it already exists.  This is useful if an object may be used by many other ones, and we want make sure that we will not inadvertedly create a copy of it if it already exists.


Examples: 


1. Instead of writing: 


w = Window.new('test');


w.close;


Write: 


\test.window;


\test.close;


2. Instead of writing: 


a = Synth('default');

a.set(\freq, 880);

a.free;


Write: 


\default.playDef;

\default.set(\freq, 880);

\default.free;
